# Homebrew Tap for Ravi CLI

This is the official Homebrew tap for [Ravi CLI](https://github.com/ravi-hq/cli), the programmatic interface for AI agent identities.

## Installation

```bash
brew tap ravi-hq/tap
brew install ravi
```

Or install directly:

```bash
brew install ravi-hq/tap/ravi
```

## Updates

```bash
brew update
brew upgrade ravi
```
